#Install Nginx
----------------------------------------------------

1. Install Nginx
dnf -y install nginx

2. Make new directories for html files, copy the files from default
location and set permissions:

mkdir -p /data/nginx/html
cp -R /usr/share/nginx/html /data/nginx/html
chown -R nginx:nginx /data/nginx/html
chmod -R 755 /data/nginx/html

3. because of selinux some settings needs to be adjusted when
changing the location of html files from default:

dnf -y install policycoreutils-python-utils
semanage permissive -a httpd_t

3. edit /etc/nginx/nginx.conf
->server_name www.aa.lan;
->root /data/nginx/html;

4. Enable Nginx and open firewall ports

systemctl enable --now nginx
firewall-cmd --add-service=http --permanent
firewall-cmd --reload

5. If and when you alter Nginx settings the daemon needs
to restarted (or reloaded):

systemctl restart nginx

6. https -> no commands chown here. See docs how to:

7. Routing from public IP is done in gateway (ns1.aa.lan)

#end TFTP configuration
--------------------------------------

1. install the tftp-server and open firewall ports for it:

dnf -y install tftp-server
firewall-cmd --add-service=tftp --permanent
firewall-cmd --reload

2. step is to create directory /data/tftpboot and copy the default tftp service files to other names:

mkdir /data/tftpboot
cp /usr/lib/systemd/system/tftp.service /etc/systemd/system/tftp-server.service
cp /usr/lib/systemd/system/tftp.socket /etc/systemd/system/tftp-server.socket


3. edit /etc/systemd/system/tftp-server.service to have following content (it is after this configuration pointing to folder /data/tftpboot):

[Unit]
Description=Tftp Server
Requires=tftp-server.socket
Documentation=man:in.tftpd
[Service]
ExecStart=/usr/sbin/in.tftpd -c -p -s /data/tftpboot
StandardInput=socket
[Install]
WantedBy=multi-user.target
Also=tftp-server.socket


4. enable the tftp service and set SELinux permissive for it:

systemctl enable --now tftp-server
systemctl daemon-reload
semanage permissive -a tftpd_t


5. install wget, make directory /data/nginx/html/linux/rocky/8.7 
and use wget to get instal-lation files of Rocky Linux from 
www.nic.funet.fi mirror:

dnf -y install wget

mkdir -p /data/nginx/html/linux/rocky/8.7
cd /data/nginx/html/linux/rocky/8.7

wget -r -nH --cut-dirs=5 --no-parent -e robots=off --reject="index.html*" https://www.nic.funet.fi/pub/mirrors/rockylinux.org/8.7/BaseOS/x86_64/os/


6. make and edit the files bios_install.ipxe and uefi_install.ipxe inside directory /data/nginx/html.
